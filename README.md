# Snake Rounds

A fast-paced snake game with rounds, power-ups, and increasing difficulty. Mobile-friendly with swipe controls.

## How to Play

- **Desktop**: Arrow Keys or WASD to move
- **Mobile**: Swipe to change direction
- **Goal**: Reach the target length before time runs out

## Coconuts

- **Brown Coconut**: Eat to grow (no speed change)
- **Green Coconut**: Speeds you up
- **Blue Coconut**: EXTREME speed boost

## Power-ups & Items

- **Ice Cube**: Slows you down
- **Gold Coins**: Collect 50 for an extra life
- **Red Barriers**: Avoid these (appear in round 2+)

## Running the Game

Simply open `index.html` in a web browser.
